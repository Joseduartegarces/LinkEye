# LinkEye
Es una aplicación web de comunicación aumentativa y alternativa (AAC) inspirada en Look to Speak. Utiliza visión artificial y seguimiento ocular (Eye-tracking) mediante la cámara frontal para permitir que personas con discapacidades motoras severas (tetraplejia, ELA) se comuniquen de forma autónoma. A diferencia de otras soluciones, integra un motor de texto predictivo en español y un sistema de selección jerárquica de letras para agilizar la generación de mensajes.

# Características
Eye-Tracking sin hardware: Funciona en cualquier dispositivo Android con cámara frontal mediante la web.
Selección Jerárquica: Sistema de filtrado de letras por cuadrantes (estilo Look to Speak) para máxima precisión.
Diccionario Predictivo: Motor de sugerencias en español que aprende de la frecuencia de uso.
Síntesis de Voz (TTS): Conversión de texto a voz instantánea al finalizar la frase.
Modo PWA: Instalable como aplicación nativa para uso offline y pantalla completa.
Privacidad: El procesamiento de la mirada se realiza localmente en el dispositivo (no se envía video al servidor).
